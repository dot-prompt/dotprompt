# Unspecified Referential Index - Practice Exercises

## Exercise 1: Recognition

**Instructions**: Identify the unspecified referential index (vague subject) in each sentence.

1. "People can learn quickly when they find the right approach."
2. "Some find it easy to let go of worry."
3. "They say that change happens naturally when you're ready."
4. "One can discover hidden resources within."
5. "Many have walked this path before."

**Answers**:
1. "People" - universal identification
2. "Some" - partial group identification
3. "They" - external reference
4. "One" - formal, detached reference
5. "Many" - quantity emphasis
