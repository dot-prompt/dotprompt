# Phonological Ambiguity - Practice Exercises

## Exercise 1: Recognition

**Instructions**: Identify the phonological ambiguity pattern (homophone with double meaning) in the following sentences.

1. "You can see clearly now."
2. "Let your mind wander."
3. "The son shines today."
4. "It might be right."
5. "You can bear with me."

**Answers**:
1. see/sea - visual clarity + ocean
2. wander/wonder - roaming + curiosity
3. son/sun - child + star
4. right/write - correct + pen
5. bear/bare - tolerate + animal
