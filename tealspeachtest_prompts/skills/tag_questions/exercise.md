# Tag Questions - Practice Exercises

## Exercise 1: Recognition

**Instructions**: Identify the tag question pattern and the implied agreement in each sentence.

1. "You're feeling good, aren't you?"
2. "This makes sense, doesn't it?"
3. "You can relax, can't you?"
4. "It's working well for you, isn't it?"
5. "You're ready to proceed, aren't you?"

**Answers**:
1. Tag "aren't you?" invites affirmation of feeling good
2. Tag "doesn't it?" invites affirmation that it makes sense
3. Tag "can't you?" invites affirmation of ability to relax
4. Tag "isn't it?" invites affirmation that it's working
5. Tag "aren't you?" invites affirmation of readiness
