# Nominalizations - Scoring Guide

## Overview
Nominalizations measure the ability to transform dynamic processes into static nouns, making change seem manipulable. The key is creating a sense of solidity that allows the listener to project personal meaning.

## Scoring Rubric

### High Score (80-100): Excellent Usage
- Effectively transforms dynamic processes into nouns
- Creates sense of solidity and manipulability
- Allows listener to project personal meaning
- Natural delivery with transformation language

**Indicators**:
- Verb-to-noun transformations present
- Language suggests the noun can change
- Listener can project their own meaning

### Medium Score (40-79): Partial Usage
- Attempts nominalizations
- May use concrete nouns instead
- May be too specific about meaning
- Some transformation language present

**Indicators**:
- Some noun forms present
- Less vagueness
- Partial effect on listener

### Low Score (1-39): No Usage
- No nominalization pattern
- Uses only concrete nouns
- Creates fixed/unchangeable state
- Explains nominalization specifically

**Indicators**:
- No verb-to-noun transformation
- Fixed states described
- No transformation language

## What to Listen For
- "Your decision," "your frustration," "the understanding"
- Transformation language ("can become," "is changing")
- Sense of a "thing" that can be manipulated

## Watch Out For
- Using concrete nouns - Reduces score
- Being too specific - Reduces score
- Creating fixed states - Reduces score
- Not allowing listener flexibility - Reduces score

## Example Assessments

### Example Response 1: "Your frustration can become curiosity."
**Score**: 90
**Reasoning**: Clear nominalization. Process (frustrated) transformed to noun (frustration). Transformation language present. Natural.

### Example Response 2: "You are frustrated."
**Score**: 25
**Reasoning**: Uses adjective form, not nominalization. No transformation possible suggested.

### Example Response 3: "Nice weather today."
**Score**: 5
**Reasoning**: No nominalization. Off-topic.

### Example Response 4: "Your frustration is a fixed state that cannot change."
**Score**: 15
**Reasoning**: Nominalization used but creates fixed state. Opposite of intended effect.
