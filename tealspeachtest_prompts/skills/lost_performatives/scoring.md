# Lost Performatives - Scoring Guide

## Overview
Lost Performatives measure the ability to state values as universal truths without attribution. The key is deleting the source of the judgment to make opinions sound like objective facts.

## Scoring Rubric

### High Score (80-100): Excellent Usage
- Values stated as universal truths without attribution
- Makes opinions sound like objective facts
- Values align with therapeutic goals
- Natural delivery

**Indicators**:
- "It's good to...," "It's important to...," "It's natural to..." present
- No attribution or source
- Values listener would likely accept

### Medium Score (40-79): Partial Usage
- Attempts lost performatives
- May add qualifiers that reveal source
- Uses values that may conflict with listener
- Pattern present but not fully effective

**Indicators**:
- Some attribution present
- Values may not align with listener
- Less natural delivery

### Low Score (1-39): No Usage
- No lost performative pattern
- Statements are preachy/moralistic
- Values conflict with listener
- Explicitly attributes the value

**Indicators**:
- Direct attribution ("I think," "Experts say")
- Preachy tone
- Creates resistance

## What to Listen For
- "It's good to..."
- "It's important to..."
- "It's natural to..."
- No attribution/source

## Watch Out For
- Preachy tone - Reduces score
- Mismatched values - Reduces score
- Qualifiers that reveal source - Reduces score
- Overusing pattern - Sounds scripted

## Example Assessments

### Example 1: "It's good to relax before sleep."
**Score**: 90
**Reasoning**: Clear lost performative. Value stated as universal truth. No attribution. Natural.

### Example 2: "I think it's good to relax."
**Score**: 35
**Reasoning**: Lost but attribution revealed ("I think"). Breaks the effect.

### Example 3: "Nice weather today."
**Score**: 5
**Reasoning**: No lost performative. Off-topic.

### Example 4: "It's important to do what I say."
**Score**: 20
**Reasoning**: Value stated but conflicts with listener autonomy. Preachy tone.
