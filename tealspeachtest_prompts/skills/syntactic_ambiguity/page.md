# Syntactic Ambiguity

Syntactic Ambiguity involves using sentence structures that can be parsed multiple ways, creating engagement while bypassing the conscious filter.

## Definition
Identifying and utilizing sentence structures that can be parsed multiple ways, creating engagement while bypassing the conscious filter.

## Core Effect
The brain attempts to parse grammatical structure while simultaneously processing meaning. This creates dual processing—while the conscious mind is figuring out the structure, the unconscious absorbs both interpretations.

## Key Components
- **Multiple Parses**: Sentence structures with multiple valid interpretations
- **Grammatical Correctness**: Both interpretations must be valid grammatically
- **Structure-Based**: Unlike phonological (sound-based), this is grammar-based
- **Engagement**: Brain stays engaged trying to resolve structure

## Types of Structures
- **Relative clause attachment**: "I saw the man with the telescope"
- **Verb phrase attachment**: "Flying planes can be dangerous"
- **Participial phrases**: "The horse raced past the barn fell"
- **Noun phrase coordination**: "Old men and women"

## Examples
- "I saw the man with the telescope." (I had telescope? Man had telescope?)
- "Flying planes can be dangerous." (Planes are dangerous? Act of flying is dangerous?)
- "They are visiting relatives." (visiting them? or being visited?)

## Common Mistakes
1. **Confusion Without Purpose**: Creating ambiguity that doesn't serve the goal
2. **Grammatical Errors**: Making sentences that don't parse at all
3. **Too Complex**: Making it so complex neither meaning parses clearly
4. **Breaking Flow**: Sounding unnatural or confusing
