# Unspecified Verbs

Unspecified Verbs involve using action verbs that don't specify how the action happens, allowing the listener to project their own process.

## Definition
Identifying and utilizing action verbs that don't specify how the action happens, allowing the listener to project their own process.

## Core Effect
When you say "you'll change," the listener's mind immediately runs through possibilities based on their own experience. This makes the suggestion feel personalized. The trick is being vague enough to allow projection but clear enough to maintain therapeutic direction.

## Key Components
- **Vague Action**: Verbs that don't specify method
- **Personal Projection**: Listener fills in their own process
- **Therapeutic Direction**: Clear about what, vague about how
- **Increased Buy-in**: Listener feels it's their own idea

## Good Verbs
learn, change, improve, get better, grow, develop, discover, understand

## Avoid
explain (requires explanation), teach (requires teacher), show (requires demonstration)

## Examples
- "You can learn quickly."
- "Things will get better."
- "You're improving every day."
- "Growth happens naturally."

## Common Mistakes
1. **Adding Too Many Specs**: Breaking the effect by explaining the method
2. **Using Specific Verbs**: Using verbs that require objects
3. **Too Vague**: Being so vague the statement loses meaning
4. **Wrong Direction**: Implying negative change
