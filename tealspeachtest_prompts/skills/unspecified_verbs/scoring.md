# Unspecified Verbs - Scoring Guide

## Overview
Unspecified Verbs are scored based on the effective use of action verbs that don't specify how the action happens. The key is allowing the listener to project their own process while maintaining therapeutic direction.

## Scoring Rubric

### High Score (80-100): Excellent Usage
- Verb implies outcome without requiring specific method
- Listener's unconscious can fill in their own process
- Therapeutic direction maintained
- Pattern effectively stacked with other Milton patterns

**Indicators**:
- Verbs like "learn," "change," "improve," "grow" used
- No specification of how
- Listener projects their own method
- Positive direction

### Medium Score (40-79): Partial Usage
- Some unspecified verbs used
- Some specification that limits projection
- Direction generally maintained
- Pattern present but not fully optimized

**Indicators**:
- Some vague verbs present
- Some "how" is specified
- Partial room for projection

### Low Score (1-39): No Usage
- Verb is too specified
- No room for personal projection
- Direction unclear or absent
- Missing Milton Model intent

**Indicators**:
- Specific methods explained
- "You'll change by..." format
- No vague action verbs

## What to Listen For
- Action verbs like "learn," "change," "improve," "get better," "grow," "develop" without "how"
- Statements that leave process open

## Watch Out For
- **Adding specifics** - Breaks projection effect
- **Specific verbs** - Requires objects/methods
- **Too vague** - Loses all meaning
- **Negative direction** - Wrong outcome implied

## Example Assessments

### Example 1: "You'll learn what you need to learn, and that learning will serve you well."
**Score**: 90
**Reasoning**: Excellent unspecified verb usage. "Learn" doesn't specify how. Listener fills in their own process. Therapeutic direction maintained.

### Example 2: "You can change and get better over time."
**Score**: 70
**Reasoning**: Some unspecified verbs used. "Change" and "get better" are vague. Some partial specification but still allows projection.

### Example 3: "You will change by going to therapy every week."
**Score**: 20
**Reasoning**: Too specified. Method is explained ("going to therapy weekly"). No room for personal projection. Breaks the effect.
